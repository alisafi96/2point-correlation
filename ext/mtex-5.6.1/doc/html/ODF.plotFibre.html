plotFibre

plot odf along a fibre

Syntax 

plotFibre(odf,f);

Input

odf ODF
f fibre

Options

resolution resolution of each plot

Example 

odf = SantaFe;
f = fibre.gamma(odf.CS,odf.SS)
plotFibre(SantaFe,f)
 
f = fibre  
 size: 1 x 1
 crystal symmetry:  m-3m
 specimen symmetry: 222
 h || r: (111) || (0,0,1)
 o1 -> o2: (180°,54.7°,45°) -> (270°,54.7°,45°)
