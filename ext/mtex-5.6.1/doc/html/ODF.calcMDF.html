calcMDF

calculate the uncorrelated misorientation distribution function (MDF) from one or two ODF

Syntax 

mdf = calcMDF(odf)
mdf = calcMDF(odf1,odf2,'bandwidth',32)

Input

odf ODF
odf1, odf2 ODF

Output

mdf ODF

Options

bandwidth bandwidth for Fourier coefficients (default -- 32)

See also

EBSD.calcODF
