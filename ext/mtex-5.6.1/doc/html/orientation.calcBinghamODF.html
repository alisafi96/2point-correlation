calcBinghamODF

calculate ODF from individuel orientations via kernel density estimation

calcODF is one of the core function of the MTEX toolbox. It estimates an ODF from a set of individual crystal orientations by kernel density estimation.

Syntax 

odf = calcBinghamODF(ori,varargin)

Input

ori orientation

Output

odf ODF
