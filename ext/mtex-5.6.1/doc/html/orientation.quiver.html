quiver

annotate a orientation to an existing plot

Syntax 

quiver(ori)
quiver(ori,'label','o1')
quiver(ori,'MarkerFaceColor','blue')
quiver(ori,'filled')

Input

ori orientation

Options

MarkerFaceColor
MarkerEdgeColor
MarkerSize
label

Flags

filled
labeled
